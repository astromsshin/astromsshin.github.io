This page is for personal usages. But feel free to find useful information in this web page.

- Pluto : Godunov-type code for relativisitc/non-relativistic hydrodynamics/magnetrohydrodynamics (ApJS, 170, 228-242, 2007 and A&A, 488, 429-440, 2008) (http://plutocode.to.astro.it/)

- GADGET-2 : The cosmological simulation code GADGET-2 (MNRAS, 364, 1105-1134, 2005) = TreeSPH N-body + hydrodynamics (http://www.mpa-garching.mpg.de/gadget/)

- TPM : Tree particle-mesh: an adaptive, efficient, and parallel code for collisionless cosmological simulation (ApJS, 145, 1-13, 2003) = TreePM (http://www.astro.princeton.edu/%7Ebode/TPM/index.html)

- Gasoline : a flexible, parallel implementation of TreeSPH (NewA, 9, 137-158, 2003) = TreeSPH N-body + hydrodynamics (http://imp.mcmaster.ca/software/)

- GCD+: a new chemodynamical approach to modelling supernovae and chemical enrichment in elliptical galaxies (MNRAS, 340, 908-922, 2003) = N-body/SPH (tree-based) + SF and so on.

- RAMSES: a parallel adaptive mesh refinement code for astrophysical fluid dynamics (A&A, 385, 337-364, 2002) = tree-based AMR, a second-order unsplit Godunov solver (http://www-dapnia.cea.fr/Projets/COAST/ramses.htm)

- HERACLES: radiative transfer + hydrodynamics in 3D and FVM (http://irfu.cea.fr/Projets/COAST/heracles.htm)

- NIRVANA+ : AMR hydrodynamics/magnetohydrodynamics (Computer Physics Communications, 109, 111-134, 1998) (http://nirvana-code.aip.de/)
